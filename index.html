Since the beginning of the year the number of API users has doubled to over 1400 API keys. That's over 5 new API keys being created per day!

So what is being built? Nobody really knows.

There is no way for people to know that there not only multiple Windows Phone 8 apps, but also Blackberry options. Developers can't get a head start on building even more Strava apps by leveraging user-built SDKs, because they can't get to them. The next budding Veloviewer.com might dwindle due to lack of exposure.

It's time to provide a place to list, incentivize, & showcase what people are doing with our API. It is time for a new http://www.strava.com/developers page.
